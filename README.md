# 💻 Felipe Gonçalves dos Santos | Cloud & Fullstack Portfolio

Este repositório contém o código-fonte do meu portfólio profissional. Este projeto foi desenvolvido para funcionar como um portal interativo e dinâmico, demonstrando de forma prática minhas competências atuais e meu direcionamento profissional para a área de Cloud Computing e DevOps.

---

## 🚀 Sobre o Projeto

Este portfólio é mais do que um simples cartão de visitas; é um ambiente interativo que reflete minha paixão por infraestrutura e arquitetura em nuvem. Ele é dividido em seções-chave:
1. **Terminal Interativo (CLI)**: Uma simulação interativa de terminal shell no navegador, onde os visitantes podem interagir digitando comandos.
2. **AWS Interactive Architecture Hub**: Um simulador visual de arquitetura multi-camadas (us-east-1) que exibe as configurações e especificações declarativas de cada serviço (como CloudFront, ALB, EC2, Lambda, RDS e S3) ao clique do usuário.
3. **Projetos em Destaque**: Portfólio de projetos cobrindo automação de infraestrutura (IaC com Terraform), Backend Serverless e Frontend.

## 📂 Estrutura do Projeto

O projeto é construído de forma leve e performática utilizando as tecnologias fundamentais da Web:
* `index.html` — Estrutura semântica das seções, terminal interativo e do AWS Hub.
* `style.css` — Estilização moderna e responsiva utilizando variáveis CSS, efeitos de vidro (glassmorphism) e animações suaves.
* `script.js` — Lógica do terminal interativo, efeito do canvas de partículas em rede, simulador interativo de infraestrutura AWS e controle do formulário de contato.

## 💻 Como Executar o Projeto Localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/DevFelipeGS/Portfolio.git
   ```
2. Acesse a pasta do projeto:
   ```bash
   cd Portfolio
   ```
3. Abra o arquivo `index.html` diretamente em seu navegador preferido ou utilize uma extensão de servidor local (como *Live Server* no VS Code).
